# User Management

Provisioners create and manage SSO users — users who authenticate via the provisioner's external identity system rather than a local password.

## Create SSO User

```http
POST /provisioner/users
Authorization: Bearer prov_sk_live_...
Content-Type: application/json

{
  "providerId": "provider-uuid",
  "externalId": "ext-12345",
  "email": "player@example.com",
  "phone": "+1-555-0100",
  "providerRole": "DIRECTOR",
  "ssoProvider": "federation-idp"
}
```

**Response (201):**

```json
{
  "success": true,
  "userId": "generated-uuid",
  "email": "player@example.com",
  "providerRole": "DIRECTOR",
  "providerId": "provider-uuid"
}
```

**What happens internally:**

1. Validates the provisioner manages the specified provider
2. Checks no existing SSO identity for this `ssoProvider` + `externalId`
3. Checks email is not already registered
4. Creates user record with **empty password** (SSO-only — cannot log in with password)
5. Creates SSO identity mapping (`ssoProvider` + `externalId` → `userId`)
6. Associates user with the provider via `user_providers` join table

## List Provider Users

```http
GET /provisioner/users?providerId=<provider-uuid>
Authorization: Bearer prov_sk_live_...
```

**Response:**

```json
{
  "users": [
    {
      "userId": "uuid",
      "email": "player@example.com",
      "providerRole": "DIRECTOR",
      "ssoProvider": "federation-idp",
      "externalId": "ext-12345"
    }
  ]
}
```

**Error codes:**
- `PROVIDER_NOT_MANAGED` — the provisioner doesn't manage this provider
- Returns `{ error: 'providerId query parameter required' }` if query param is missing

## SSO User Properties

- **No password**: SSO users have an empty password hash — they cannot authenticate via the standard login endpoint
- **Identity-driven**: Authentication happens through the SSO token flow (see SSO documentation)
- **Provider-scoped**: Each user is associated with one or more providers via `user_providers`
- **Email uniqueness**: Enforced globally — an email can only exist once across all providers
