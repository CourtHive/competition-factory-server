<h2>Quick Start</h2>

<pre style={{ background: '#f5f5f5', padding: '16px', borderRadius: '6px', fontSize: '13px', overflow: 'auto' }}>
{`# Clone and install
git clone https://github.com/CourtHive/competition-factory-server.git
cd competition-factory-server
pnpm install

# Start the database (LevelDB)
pnpm hive-db

# Create an admin user (in another terminal)
pnpm admin:create -e admin@example.com -p your-password

# Start the dev server
pnpm watch`}
</pre>
