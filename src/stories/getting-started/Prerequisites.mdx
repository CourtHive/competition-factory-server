## Database (choose one)

### Option A: LevelDB (default)

No separate installation needed. The server includes `@gridspace/net-level` which runs as a child process.

```bash
# Start the LevelDB server
pnpm hive-db
```

This starts a LevelDB-over-network server on port 3838. Data is stored locally in the `data/` directory.

### Option B: PostgreSQL

```bash
# macOS (Homebrew)
brew install postgresql@17
brew services start postgresql@17

# Create the database
createdb courthive
```

See the **Storage / PostgreSQL Setup** page for full configuration details.

## Redis

Redis is required for server-side caching.

```bash
# macOS (Homebrew)
brew install redis
brew services start redis

# Verify it's running
redis-cli ping
# → PONG
```

Default configuration connects to `redis://localhost:6379`. Override with `REDIS_URL` in `.env`.

## Optional: Email (Mailgun)

For user invitation emails, you need a Mailgun account. Set `MAILGUN_API_KEY`, `MAILGUN_DOMAIN`, and `MAILGUN_HOST` in `.env`. The server works without email — invitations will simply not be sent.

## Verify Setup

```bash
node --version    # v22.x.x or higher
```
