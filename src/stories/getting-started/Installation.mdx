# Installation

## 1. Clone the repository

```bash
git clone https://github.com/CourtHive/competition-factory-server.git
cd competition-factory-server
```

## 2. Install dependencies

```bash
pnpm install
```

> **Note:** `npm` is not supported — use `pnpm` only.

## 3. Configure environment

Copy the example `.env` file and customize:

```bash
cp .env.example .env
```

At minimum, set a secure JWT secret:

```env
JWT_SECRET='your-truly-random-secret-string-here'
```

See the **Configuration** page for the full environment variable reference.

## 4. Start the database

**LevelDB (default):**

```bash
pnpm hive-db
```

This starts the LevelDB network server on port 3838. Keep this running in a separate terminal.

**PostgreSQL:**

See **Storage / PostgreSQL Setup** for schema creation and migration steps.

## 5. Create an admin user

In a new terminal (with the database running):

```bash
pnpm admin:create -e admin@example.com -p your-secure-password
```

This creates a superadmin user with full access.

## 6. Start the server

**Development (with watch mode):**

```bash
pnpm watch
```

**Production:**

```bash
pnpm build
pnpm start
```

The server starts on the port defined by `APP_PORT` in `.env` (default: `8383`).

## 7. Verify

```bash
curl http://localhost:8383/api
# Should return a response (or 404 for undefined root)
```
